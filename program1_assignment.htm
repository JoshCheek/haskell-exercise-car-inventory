What this program does: 

/* This program aids a car business in managing its inventory.
 * This program reads in file inventory.data and
 * sorts the information.
 * It runs interactively during a business day in response to
 * the needs of the sales personnel.
 */
It will need a menu for the user to choose from, and only exits when the user selects 8 (Quit) 
   1  -  Add a new car to inventory.
   2  -  Remove a sold car from inventory.
   3  -  Display info on car. (inv # needed)
   4  -  Display cars of specific Year.
   5  -  Display cars within a price range.
   6  -  Display cars of specified color.
   7  -  Display All Cars in inventory.
   8  -  Quit. (exit inventory tracking system)
Remember to break your program into pieces (e.g. make sure that you are reading in the data file correctly, before you start sorting, adding, deleting, etc from it in memory)

The output is ONLY to the screen/monitor in this program. This means that all the changes are done solely in memory, and the actual inventory.data file is never actually changed. (In future programs, I will ask you to either output to another file, or to overwrite the inventory.data file ... don't worry about that in this program.)

Use the inventory.data file for your input (it is attached with this assignment).

You are required to divide your program into multiple files. You need to keep your array_list files seperate from your program1 main files. I will provide you with what you need for this program ... in terms of file names and what you should have in the files ... future programs, I assume that you can infer what should go into your files. Here is the list of files you should have:

  1. program1.c your main() code and function definitions not related to array_lists (e.g. void remove_null(char *s))

  2. program1.h your function declarations/headers that you have defined in program1.c

  3. prog_defs.h your program definitions (macros, structures, #defines), but not specifically the array_list structure

  4. array_list.c your array_list function definitions

  5. array_list.h your array_list function declarations, your array_list structure

I have attached the inventory.data file, the prog_defs.h file (you will need to alter this for future programs).

I have attached the starting shell of array_list.c and array_list.h files. You will need to add any additional or altered function declarations and definitions that you deem necessary to complete the tasks of the program (e.g. possibly an altered sequential search function that handles ONLY looking for color, or a print_record function).

Please notice the #include statements at the top of the files. You will need to do similar in future programs (e.g. linked lists).

Please review Tips in Debugging your Programs under the Course Tools and Help section.
